## Featured Projects

### [32-Bit Kogge-Stone Adder](https://github.com/upadhyaypranjal/32-Bit-Kogge-Stone-Adder)
Complete RTL-to-GDSII implementation of a high-speed parallel prefix adder, synthesized and physically implemented across both **90nm** and **180nm** CMOS nodes. Logarithmic carry-propagation delay, fully timing-clean (DRC/LVS pass) in both technologies.
**Stack:** Verilog · Cadence Genus · Cadence Innovus

### [32-Bit RCA — DFT & ATPG](https://github.com/upadhyaypranjal/RCA32-DFT-and-ATPG)
Scan-based Design-for-Testability architecture built on a sequential 32-bit Ripple Carry Adder. Single muxed-scan chain across 98 flip-flops, zero DFT rule violations, and **99.95% static fault coverage** via Cadence Modus ATPG.
**Stack:** Verilog · Cadence Genus · Cadence Modus

### [FPGA Multi-Protocol Analyzer / Sniffer](https://github.com/upadhyaypranjal/FPGA-based-Multi-Protocol-Analyzer-Sniffer)
Hardware logic analyzer that decodes UART, I²C, and SPI traffic directly in FPGA fabric in real time, streams decoded packets to an RP2040 over SPI, and renders them live in a PyQt6 desktop GUI — byte-by-byte, protocol-tagged, with zero software polling overhead.
**Stack:** Verilog · ForgeFPGA · RP2040 · PyQt6

### [FPGA Secure Image Transfer — BFV Encryption](https://github.com/upadhyaypranjal/FPGA-Based-Secure-Data-Transfer-of-Images-with-Cloud-Using-BFV-Homomorphic-Encryption)
Hardware-accelerated homomorphic encryption pipeline (Xilinx Zynq-7000) that encrypts images with BFV FHE before they ever leave the edge device, then transfers and verifies them through a cloud pipeline with 100% pixel-perfect recovery. Also validated through a full ASIC RTL-to-GDSII flow at 180nm and 90nm.
**Stack:** Verilog · Vivado/Vitis · Cadence Genus/Innovus · Python
*Team project with Divyansh Tiwari, Vikramaditya Singh & Sohan Maity — IIITDM Kurnool*
