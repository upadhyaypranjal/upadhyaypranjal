## 🚀 Featured Projects

### ⚡ 32-Bit Kogge-Stone Adder
Complete RTL → GDSII implementation of a high-speed parallel-prefix adder.

- 90nm & 180nm CMOS
- Logarithmic carry architecture
- Timing clean • DRC passed • LVS passed

`Verilog` `Cadence Genus` `Cadence Innovus`

🔗 [Repository](https://github.com/upadhyaypranjal/32-Bit-Kogge-Stone-Adder)

---

### 🧪 32-Bit RCA — DFT & ATPG
Scan-based Design-for-Test architecture built on a sequential 32-bit Ripple Carry Adder.

- 98 scan flip-flops
- Single muxed scan chain
- Zero DFT rule violations
- 99.95% static fault coverage

`Verilog` `Cadence Genus` `Cadence Modus`

🔗 [Repository](https://github.com/upadhyaypranjal/RCA32-DFT-and-ATPG)

---

### 🔍 FPGA Multi-Protocol Analyzer / Sniffer
Real-time FPGA protocol analyzer decoding UART, I²C, and SPI — packets stream to an RP2040 over SPI and are visualized live through a PyQt6 desktop application.

`Verilog` `ForgeFPGA` `RP2040` `PyQt6`

🔗 [Repository](https://github.com/upadhyaypranjal/FPGA-based-Multi-Protocol-Analyzer-Sniffer)

---

### 🔐 FPGA Secure Image Transfer using BFV Homomorphic Encryption
Hardware-accelerated image encryption pipeline using BFV Fully Homomorphic Encryption.

- FPGA acceleration
- Secure cloud transfer
- Pixel-perfect recovery
- ASIC validation at 90nm & 180nm

`Verilog` `Vivado` `Vitis` `Cadence Genus` `Cadence Innovus` `Python`

🔗 [Repository](https://github.com/upadhyaypranjal/FPGA-Based-Secure-Data-Transfer-of-Images-with-Cloud-Using-BFV-Homomorphic-Encryption)
